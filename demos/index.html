Media queries

Mobile-first queries are how the grid is built and is the preferrable method of organizing CSS. It means all styles outside of a query apply to all devices, then larger devices are targeted for enhancement. This prevents small devices from having to parse tons of unused CSS. The sizes for the queries are:

  • Large desktop screens: 112.5em
  • desktop screens: 75em
  • tablet lanscape: 56.25em
  • tablet portrait - up: 37.5em
  • phone: ~up untill 37.4375em

Typography

Type is all set with the rems, so font-sizes and spacial relationships can be responsively sized based on a single <html> font-size property./p>

The typography base is Open Sans served by Google, set at 15rem (15px) over a 1.6 line height (24px). Other type basics like anchors, strong, emphasis, and underline are all obviously included.

Don't forget blockquotes! Blockquotes have a simple left border and an indent, there is nothing too fancy here. These styles are meant to be familiar to GitHub markdown users.

Headings create a family of distinct sizes each with specific letter-spacing, line-height, and margins.
