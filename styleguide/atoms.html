What are Atoms?
  • Form elements
    • Buttons
Atoms are the very basic elements of the stylesheet. Imagine every single-class element or selector rules as an atom. Examples could be buttons, links, headlines, inputs ... you get it, right?
To mark an element as an atom, annotate it with the @atom tag, followed by the name of the component. Special atoms are @icons.

Button Styles Modifiers

Modifier
Description
.{prefix}-button
default style for button (gray color).
.--primary
button that contains primary teme color
.--secondary
button that contains secondary teme color
.--error
button that contains (ususally) a red color
.--proceed
button that contains (ususally) a green color
.--disabled
used on buttons that are no longer clickable (disabled)
.--alternative
